NoteFusion AI - Node.js Setup Guide

1. Install Node.js

It looks like Node.js might not be properly installed or configured on your system.

  1. Download Node.js LTS from: https://nodejs.org/
  2. Run the installer
  3. Make sure to check "Automatically install the necessary tools" during installation
  4. Complete the installation
  5. Restart your computer after installation
Download Node.js

2. Verify Installation

After restarting your computer, open a new Command Prompt and run:

node -v
npm -v

You should see version numbers for both Node.js and npm if they are installed correctly.

3. Start the Application

Once Node.js is installed, open Command Prompt and run these commands:

cd c:\Users\User\notefusion-ai\notefusion-ai\frontend
npm install --legacy-peer-deps
npm run dev

The application should open automatically in your browser at http://localhost:3000

4. Need Help?

If you encounter any issues:

  • Make sure you've restarted your computer after installing Node.js
  • Check that Node.js is in your system PATH
  • Take a screenshot of any error messages
  • Contact support with the error details
