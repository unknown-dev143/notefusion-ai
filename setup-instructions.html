NoteFusion AI Setup Instructions

1. Check Node.js Installation

Let's verify if Node.js is installed on your system:

Open Command Prompt and run: node -v

If you see a version number (like v22.18.0), Node.js is installed.

If you see an error, please install Node.js from the link below.

2. Install Node.js (if needed)

If Node.js is not installed, please:

  1. Download Node.js LTS from: https://nodejs.org/
  2. Run the installer
  3. Make sure to check "Automatically install the necessary tools" during installation
  4. Complete the installation
  5. Restart your computer after installation
Download Node.js

3. Start the Application

After installing Node.js, open Command Prompt and run these commands:


cd c:\Users\User\notefusion-ai\notefusion-ai\frontend
npm install --legacy-peer-deps
npm run dev

The application should open automatically in your browser at http://localhost:3000

4. Need Help?

If you encounter any issues, please:

  1. Take a screenshot of any error messages
  2. Share the error messages when you ask for help
  3. Make sure you've restarted your computer after installing Node.js
